简介

故事背景

玩家将在游戏中探索一个被称作「提瓦特」的幻想世界。在这广阔的世界中，被神选中的人将被授予「神之眼」，称为「原神」，引导元素之力。

因故国毁灭，而在无数世界中漂流的兄妹，准备离开幻想世界“提瓦特”前往下一个世界时，遭陌生神灵“‘天理’的维系者”阻拦，其中一位被神灵带走，另一位失去力量陷入沉睡，并在不知多少年后苏醒，开始于提瓦特流浪。在流浪途中，派蒙被旅行者从河中钓起，开始担任旅行者的向导。旅行者学习了提瓦特的语言，就此开始旅行。

玩家将扮演「旅行者」，可以踏遍七国，邂逅性格各异、能力独特的同伴，与他们一同对抗强敌，踏上寻回血亲之路；也可以不带目的地漫游，沉浸在充满生机的世界里，让好奇心驱使自己发掘各个角落的奥秘……直到你与分离的血亲重聚，在终点见证一切事物的沉淀，同时，逐步发掘「原神」的真相。

玩法介绍

游戏类型为动作角色扮演游戏。玩家的基本操作有移动、攻击、闪避/冲刺、施放元素战技或元素爆发、交互或拾取、攀爬垂直表面、游泳、使用基本功能（如派蒙主菜单、聊天室等等）、空中飞行等等。玩家通过完成任务与祈愿（抽卡）系统，将可以解锁超过30位角色。每一位可操纵角色都是可以养成的，能够实质改变角色性能的系统包括角色等级，圣遗物搭配，武器搭配，天赋等级，命之座等。

探索机制

玩家存在两个等级：冒险等阶和世界等级。当冒险等阶达到一定的水平后，会提升世界等级。世界等级主要会影响到世界敌人的强度，掉落物的品质，有时也会影响角色养成的上限。玩家达到冒险等阶16级后可开启联机，联机模式下至多四名玩家可以共同进行游戏，玩家可以添加好友一同交流或联机游玩。

玩家一次可以选择四名角色（联机模式下所有玩家总共四名角色）上场游玩。角色在进行高强度运动时，将会消耗体力值，施放重击（使用弓作为武器的角色除外）、闪避、持续冲刺、游泳、攀爬、快速攀升以及展开风之翼滑翔均会消耗体力值，若体力值不足，将无法进行上述动作。当玩家持续站在陆地上行走、慢跑或站立时，延迟0.5秒后持续恢复体力值。可用的体力值会在游戏界面中间偏右的位置显示为圆弧型的进度条，每满100体力值将划分一小格，不足100体力值的部分同样划分一个小区域。

完成“魔神任务序章 第一幕：「自由之都」”任务后开启飞行系统。展开风之翼滑翔可减缓角色的空中移动速度（包括下坠速度和受风圈影响后的空中加速度），玩家还可以在滑翔期间用方向键调整滑翔方向。

世界传送点包括七天神像、传送锚点、秘境入口等，玩家在地图上选择传送点对角色进行传送。七天神像分为火、水、风、雷、草、冰、岩七种元素类型，每种元素类型的七天神像有若干座，散布在对应元素地区的不同区域内。激活七天神像可以点亮对应区域的地图。一次激活永久有效。首次激活一种元素类型的七天神像，旅行者即可获得对应元素属性的天赋和命之座。旅行者通过与不同元素属性的神像共鸣，可切换旅行者的元素属性。当角色进入七天神像交互判定范围后，可消耗神像所存储的恢复力以补充角色生命值。神像间共享所储存的恢复力，恢复力存储总上限随玩家解锁的神像数而提升。当恢复力未达到神像恢复力存储总上限时，会随时间进行补充。使用「口袋锚点」道具可在世界任意位置处放置至多一个且持续一周的临时传送锚点。

在世界中存在可以用摩拉货币购买物品的各种商店。玩家可以利用商店购买食谱、食材、材料、摆设图纸、小道具、低级圣遗物和武器等。在地区声望系统达到一定等级时会开启地区商店折扣。在一些活动中有活动商店，玩家可以利用活动代币在活动商店中兑换珍贵物品和限定小道具。活动代币会在活动商店关闭时过期。商店在刷新周期内拥有一定的限购量。食谱、小道具、书籍、摆设图纸等只能购买一次。食物每天刷新，食材每周一刷新。纪念品商店中的商品不会刷新。玩家无法出售任何物品。玩家可通过各种途径获得食物或解锁食物图谱烹饪食物。使用一些食物可以恢复/复活自己角色的生命值或短暂的时间内增强自己队伍的属性。可在合成台将三个同类的低级素材合成一个高级素材。

完成“魔神任务第一章 第三幕、世界任务「翠石砌玉壶·其一」”后获得「尘歌壶」道具，通过「尘歌壶」系统，玩家可以制作各种摆设道具或建筑。在开启「尘歌壶」系统并完成世界任务「鱼群大爆发」后可开启钓鱼系统。

战斗系统

玩家在战斗中可以快速切换四个上场角色，一旦进入战斗状态则无法更改上场角色。游戏内拥有七种元素类型，分别为火、水、风、雷、草、冰、岩。每种元素会产生不同的元素效果，各种元素之间还会产生各种元素共鸣和。通过攻击敌人或魔物到一定进度、有效使用元素战技（包括使用元素战技命中敌人、在元素战技生效期间战斗），玩家可为角色积攒元素微粒以收集元素能量。每个角色都有两种技能——元素战技与元素爆发，元素战技只要招式的冷却时间结束就可以发动，元素爆发的施放需要累积元素能量，并皆以元素形式施放。此外，对于使用弓作为武器的角色，还有单独作为一种攻击方式的瞄准模式。元素战技、元素爆发以及部分普通攻击都可以为敌人或魔物附着元素，敌人或魔物的部分攻击以及角色天赋会为角色附着元素；世界的地理和气候环境也会为所有人附着元素。当战斗中敌人被玩家拉扯太远的时候，他们会撤回自己的基地，并无视玩家行动回满状态。除某些特殊剧情外，我方NPC角色不会协助玩家作战。即使敌人不是同一类型的，他们也不会内战，只会攻击玩家。

元素反应列表

雷 + 水 = 感电 （对目标持续造成雷元素伤害，可连锁至附近具有水元素附着的目标，每次伤害均有较高打断效果）
雷 + 冰 = 超导 （对目标及其周围造成小范围冰元素伤害并降低目标物理抗性）
火 + 雷 = 超载 （造成爆炸效果击飞目标并造成火元素范围伤害）
火 + 水 = 蒸发 （元素反应本身没有伤害但是会按比例提升本次攻击的伤害）
火 + 冰 = 融化 （元素反应本身没有伤害但是会按比例提升本次攻击的伤害）
火 + 草 = 燃烧 （对目标持续造成火元素伤害）
冰 + 水 = 冻结 （被冻结的目标将暂时无法行动，被冻结的敌人受到重击或岩属性伤害时会发生碎冰，受到碎冰伤害。）
风 + 四元素之一 = 扩散 （造成对应后者元素的伤害并将该元素大范围附着在敌人身上）
岩 + 四元素之一 = 结晶 （产生对应后者元素属性的结晶护盾，护盾对该元素有250%的吸收效果）

水面环境 + 雷 = 水面感电 （会使小范围水面产生感电并对范围内目标持续造成雷元素伤害）
草地环境 + 火 = 草地燃烧 （会使小范围草地着火并对范围内目标持续造成火元素伤害）
草地燃烧 + 雷 = 草地超载 （会产生爆炸击飞范围内目标并造成火元素伤害，同时爆炸部分草地停止燃烧）
草地燃烧 + 风 = 燃烧扩散 （会造成大范围火元素伤害）

副本分为炼武秘境、祝圣秘境、精通秘境、征讨秘境、地脉之花、深境螺旋等，通关秘境可收取奖励。除一些挑战秘境外，其余秘境收取奖励时均需要消耗树脂。原粹树脂可以激活秘境深处的石化古树，以及世界上的各种地脉之花，以获得秘宝。秘境入口具有传送功能。在秘境中还可能出现随机的地脉异常事件。

收费机制

游戏付费点主要来自祈愿和服装购买，以及收费月卡——空月祝福和充值纪行。玩家可使用现实货币兑换创世结晶，以兑换为通用货币——原石或购买服装。原石还可通过游戏任务和成就获取，部分服装可通过限时活动获取。

祈愿活动分为常驻祈愿、初行者推荐祈愿、角色活动祈愿、武器活动祈愿共四种（后两者合称为活动限定祈愿），玩家可通过祈愿获取角色以及武器，而祈愿所需道具有原石兑换、创世结晶兑换、完成游戏任务获取等途径。抽出四星/五星品质的内容会重新计算对应品质的保底计数，所有祈愿活动的保底计数会继承到下一期同类祈愿上，不会因祈愿入口关闭而被重置。在角色活动祈愿中，玩家至多在第10次祈愿时获得四星及以上角色，至多在第90次祈愿时获得五星角色（即俗称的“小保底”）。在武器活动祈愿中，玩家至多在第10次祈愿时获得四星及以上武器，至多在第80次祈愿时获得五星武器。
