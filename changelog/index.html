更新日志

此页面记述了本站的更新计划、更新历史和数据统计。
其他站点更新历史，参见 @adminbear@sn.emiliabear.com 。

更新计划

优先

  • 恩格斯《家庭、私有制和国家的起源》笔记
  • 《原神》大世界的路网研究

不急着做

  • 尽可能搜集目前互联网上没有的有关中国游戏方面的老文章

长期维护

  • 提升中国大陆地区访问速度
  • CSS样式改进

 

更新历史

2025-07-11

【改动】
主题更新至 Butterfly 5.4.2。
调整站点主要首选字体为 Noto Sans SC，文章首选字体为 霞鹭文楷屏幕阅读版GB 。
留言板和关于合并为一个页面。
评论系统 Waline 全面更新至 v3，重新适配样式，更新了提醒邮件模板。
【移除】
移除字体 Noto Sans serif。

2025-06-19

【新增】
引入 Umami 数据统计支持。
下方页码现在可以跳转了。
【改动】
Hexo 更新至 7.3.0。
主题更新至 Butterfly 5.3.5 并更新配置文件。
重构文章分类和标签。
搜索插件由 hexo-generator-search 更换为 hexo-generator-searchdb 。
调整第三方CDN源为 cdnjs 。
【移除】
下线部分转载文章和一些测试页面，转载文章的评论转移至留言板。
根据 Hexo 更新要求，移除链接至各类视频媒体网站的短代码，暂不以插件形式补回。
因使用频率很低，移除 Akilar 的 Tag Plugins Plus 插件，相关说明文章一并下线。

2023-12-15

【改动】
主题更新至 Butterfly 4.11.0。
评论系统 Waline 版本固定在 v2.15.8 以防即将更新至 v3 的 Waline 未适配主题导致页面出错。
【移除】
因使用频率低，删除 Qexo 及说说页面。
暂停 Telegram 频道更新，移除相关链接。

2023-05-21

【改动】
Hexo 更新至 6.2.0。
主题更新至 Butterfly 4.8.5。
【移除】
资源目录 <share.emiliabear.com> 下线。
【提醒】
<sn.emiliabear.com> 转为 GoToSocial 站点，并上线新账户 @adminbear@sn.emiliabear.com 专门记录其他站点更新，故本页面转为仅记录本站的更新历史。

2023-04-11

【改动】
主题更新至 Butterfly 4.8.1。隐藏了顶部图，使信息展示更为直观。

2022-11-17

【改动】
主题更新至 Butterfly 4.5.1 ，引入 pace.js 加载动画条。
Qexo 说说功能由新的 Pleroma 站点 <sn.emiliabear.com> 取代，原页面保留作为备份直到不再使用 Qexo 为止。
资源目录 <share.emiliabear.com> 转移至新的云端硬盘 <file.emiliabear.com> ，原资源目录仍将保留一段时间。
【移除】
因功能重复且无必要，下线 Bangumi 追番页面。

2022-09-30

【新增】
引入了littlefoot.js，调整脚注样式，可以查看弹出的悬浮脚注了。
为 Markdown 渲染器新增了合并单元格功能。
【改动】
调整第三方CDN源为 UNPKG 。

2022-09-19

【新增】
主题更新至 Butterfly 4.3.1 ，修改了站点部分元素的颜色，优化了部分页面的显示。
【改动】
为分类和标签页面新增了导航栏，上线了 Qexo 的说说功能。资源目录 <share.emiliabear.com> 重新上线。
状态页面改为由 Uptimerobot 提供支持。
【问题】
移动设备特定浏览器查看网页时出现背景图片时大时小问题，经测试后认为影响不大，决定暂不解决。

2022-05-18

【新增】
上线了 Bangumi 追番页面。
【改动】
主题更新至 Butterfly 4.2.1 ，优化字体和界面。
因 cdn.jsdelivr.net 在中国大陆访问状况恶化，更换了部分引用文件的CDN，并清理了一些重复的CDN设置和高度依赖原CDN的插件。

站点终于建设完成啦！

2022-03-12

【改动】
更换字体为思源宋体、思源黑体（简繁中文）和 ABeeZee ，优化了界面，调整了分类。
【移除】
因服务器问题，暂时下线文件仓库、资源目录。
因没有相关需求，不再支持PWA。

2022-01-25

【改动】
主题更新至 Butterfly 4.0.1 ，微调字体。
【移除】
因DisqusJS访问体验依然不佳，移除第二套评论系统DisqusJS。
移除 HexoPlusPlus 的说说功能，移除豆瓣页面和 bilibili 追番追剧页面。

2021-10-29

【改动】
主题更新至 Butterfly 3.8.4。
图床域名改为 <lain.emiliabear.com>。

2021-08-29

【新增】
文件仓库、资源目录正式上线。
支持PWA。增加 Aplayer 等媒体播放器支持。增加了音乐记录页面，但未完善。
增加了 HexoPlusPlus 服务，改用其提供的说说和编辑功能。
网站状态页面试运行。增加了 bilibili 追剧页面（需直接输入网址路径）。
更新日志页面增加了数据统计部分。
【改动】
优化代码、优化文件结构。移除 Volantis 主题支持。
网站的第二套评论系统由Gitalk改为DisqusJS。
更换了部分图片的图床为阿里云OSS，图床域名为 <img.emiliabear.com> 。
【移除】
移除了 Artitalk。

2021-08-15

【改动】
更换了网站域名为 <www.emiliabear.com> 。
Hexo更新为 5.4.0。
主题更新至 Butterfly 3.8.3。
【问题】
Artitalk 会莫名其妙地修改页面名称，在找到解决办法前暂时停用。

2021-08-06

【新增】
由 OneManager 搭建的资源目录（<share.emiliabear.com>）试运行，并投入到小型资源分享中。

2021-06-30

【新增】
搭建了 OneIndex，文件仓库试运行，不过最初不会放入太多文件，域名为 <cloud.emiliabear.com> 。
由于 OneIndex 搭在了 CloudFlare Workers 上，域名的DNS服务器也顺便变更到了 CloudFlare 。

2021-05-31

借助 Github Pages 和 Vercel 的服务，添加了自定义域名的解析。
添加了 Artitalk 至“留言板”页面。

2021-05-21

实现了 Vercel 自动部署，完善了 Waline 评论系统，添加了豆瓣页面。

2021-05-18

站点主题更换至 Volantis 4.3.1（仅在本地测试），后更换至 Butterfly 3.7.7，同一天完成了 Github Actions 自动部署。
新增了“留言板”和“更新记录”页面。

2021-05-15

站点主题更换至 NExT 7.8.0，随后更新至 8.4.0 。
在此期间站点恢复了之前的一篇文章，复刻了一个 Markdown 语言语法指南，已初具规模，并对外开放。

2021-05-14

开始进行建立站点的实践，在当天晚上成功在本地建立 Landscape 主题的默认内容站点。

2021-04-29

开始学习建立 Hexo 框架站点的相关知识，并在本地开始建站尝试。

2020-12

受到网络上优秀的博客影响，重新计划建立个人站点。
由于没有服务器，再次选择 Github Pages，不过此时已改用另一个账号，因此域名随之变更。

2020-03-24

建立了 Telegram 频道。

2016-05

因课业和个人懒惰等原因，删除了站点所有内容，取消了重定向。打开站点时会出现默认的404界面。
在此期间共写下20余篇文章，以个人日记为主，还有两三节短篇小说，只有几篇得到备份。现在想来有些可惜。

2016-02-16

站点依托 Github 服务器正式建立，并重定向至一个链接。
同日，创建第一篇文章。
